# Star-Wars-NanoGPT-Model
A NanoGPT model trained using Star Wars scripts. \
Kaggle Dataset: [Star Wars Movie Scripts](https://www.kaggle.com/datasets/xvivancos/star-wars-movie-scripts)

Python version used: `3.10.9`

## Library Installations
`requirements.txt` file:
```txt
gymnasium==0.28.1
ipykernel==6.19.2
numpy==1.23.1
seaborn==0.12.2
scikit-learn==1.3.0
tensorflow==2.10.0
tensorflow-datasets==1.2.0
transformers==4.29.2
torch==2.0.0
protobuf==3.20.3
streamlit
```

## How to run the app in Streamlit
1. Go to file directory and type:
```bash
cmd
code .
```

2. Go to terminal and type this command
```bash
pip install -r requirements.txt
streamlit run app.py
```
